2. OBJETIVO GERAL

O IntegrativoApp é uma plataforma digital que conecta profissionais da saúde (psicólogos, terapeutas, coaches) a seus pacientes, permitindo o agendamento de sessões online, gestão financeira e armazenamento seguro de prontuários em nuvem (Dropbox).

3. ESCOPO FUNCIONAL

3.1 Módulo de Profissionais

  • Cadastro e autenticação de profissionais
  • Configuração de agenda (horários disponíveis)
  • Definição de preço por sessão
  • Gestão de pacientes ativos
  • Dashboard financeiro (receitas, pagamentos pendentes)

3.2 Módulo de Pacientes

  • Cadastro e autenticação
  • Agendamento de sessões
  • Histórico de atendimentos
  • Prontuário digital (acesso controlado)

3.3 Módulo de Pagamentos

  • Integração com Stripe (modo teste/produção)
  • Suporte a cartão de crédito e PIX
  • Webhooks para confirmação automática
  • Histórico de transações
  • Notificações por email (mock)

3.4 Armazenamento em Nuvem

  • Sincronização com Dropbox
  • Backup automático de dados
  • Acesso offline com sincronização posterior

5. TECNOLOGIAS UTILIZADAS

Frontend:HTML5, CSS3, JavaScript (Vanilla)
Backend:Python 3.9+ (Flask)
Banco de Dados:SQLite (embarcado) + Dropbox Sync
Gateway Pagamento:Stripe API v2025-01-27
Armazenamento:Dropbox API v2

6. DEPENDÊNCIAS EXTERNAS

  • Conta ativa no Dropbox (para sincronização)
  • Conta ativa no Stripe (para processamento de pagamentos)
  • Conexão com internet para sincronização inicial
  • Navegador atualizado (Chrome, Firefox, Edge)

7. RESTRIÇÕES E PREMISSAS

Restrições:

  • Versão de teste não processa pagamentos reais
  • Requisições Stripe limitadas a 100/teste/dia
  • Armazenamento Dropbox limitado ao plano gratuito (2GB)

Premissas:

  • Usuários possuem conhecimento básico de navegação web
  • Dispositivos com suporte a JavaScript
  • Email válido para notificações
