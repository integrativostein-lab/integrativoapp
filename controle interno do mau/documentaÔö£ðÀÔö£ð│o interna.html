🚀 Roteiro de conclusão · próximas ações
FASE 1 (2 semanas)
🔐 Conectar gateways reais + WhatsApp + certificado digital
Substituir mocks por SDK oficiais (Stripe, PagSeguro, Evolution API). Implementar webhooks de confirmação e fila de notificações. Habilitar emissão de NF com certificado A1 (ICP-Brasil).
FASE 2 (1 semana)
🛡️ Segurança e auditoria completa
Criptografar dados sensíveis (CPF, CNS, histórico de saúde) com AES-256. Adicionar blacklist de tokens JWT e rate limiting granular. Revisar permissões nos endpoints críticos.
FASE 3 (2 semanas)
🧪 Testes e documentação
Construir suite de testes (Jest/supertest) para rotas de assinatura, agendamento e financeiro. Gerar documentação OpenAPI interativa. Criar script de backup automático criptografado.
FASE 4 (1 semana)
🐳 Preparação para produção escalável
Dockerfile + docker-compose, health checks, variáveis de ambiente seguras, workflow GitHub Actions para deploy contínuo. Configurar ambiente de staging.
